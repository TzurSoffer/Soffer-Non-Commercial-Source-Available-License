# Soffer-Non-Commercial-Source-Available-License
A custom License designed for apps with a UI that protects against tampering with attribution and commercial use

## Goals / Terms
This license is intended for source-available software with a visible user interface. It allows users to view, modify, and redistribute the Software for non-commercial purposes while preserving attribution to the Original Author and preventing unauthorized commercial use. It prevents people from removing credit from your project by requiring attribution to remain clearly visible to users, while still allowing others to learn from, customize, and share the Software within the limits of the license. Two versions are provided: **V1.0**, which requires attribution to remain exactly as originally presented, and **V1.1**, which allows the attribution wording to be changed to equivalent forms (such as "Originally made by" or "Forked from") as long as clear credit to the Original Author is maintained.
